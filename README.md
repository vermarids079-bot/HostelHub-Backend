# 🏠 HostelHub Backend

Backend REST API for HostelHub Smart Hostel Management System.

---

# 🚀 Features

- User Authentication
- JWT Authorization
- Student Management
- Room Management
- Booking APIs
- Leave APIs
- Complaint APIs
- Notice APIs
- Analytics APIs

---

# ⚙️ Tech Stack

- Node.js
- Express.js
- MongoDB Atlas
- JWT
- Mongoose

---

# 🚀 Installation

Clone

```bash
git clone https://github.com/vermarids079-bot/HostelHub-Backend.git
```

Install

```bash
npm install
```

Run

```bash
node server.js
```
